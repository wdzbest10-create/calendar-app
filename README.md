# カレンダーアプリ

## 概要

本アプリは、React・TypeScript・Next.js を用いて開発したカレンダーアプリケーションです。
月表示・週表示・日表示に対応し、イベントの作成・編集・削除、繰り返し予定、検索・フィルタリング、ドラッグ操作による日付変更などを実装しました。

## 工夫した点

### 1. Feature First 構成

calendar 機能を中心に構成し、関連ファイルを集約しました。

### 2. Hook と Utility の分離

hooks には状態管理・UIロジックを配置し、lib には純粋関数を配置することで責務を分離しました。

### 3. 繰り返しイベントの例外管理

繰り返し予定に対して個別編集・個別削除ができるよう exceptions を実装しました。

### 4. ドラッグ操作の実装

イベント移動やリサイズを実装し、Google Calendar に近い操作感を目指しました。

### 5. TypeScriptによる型安全性

CalendarEvent、RepeatRule、PositionedEvent などを型定義し、保守性を向上させました。
